# Security Policy

## Supported Versions

Use laravel 11 and php 8.3

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

  If you found some error create a new bug issue to fix soon as possible.
